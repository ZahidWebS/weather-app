A simple and responsive weather app built using **HTML, CSS, and JavaScript**.

## Features
- Search for any city to get current weather
- Displays temperature, humidity, wind speed, and weather description
- Shows weather icon based on current conditions
- Clean and responsive design for desktop and mobile

## Technologies Used
- HTML
- CSS
- JavaScript
- Optional: OpenWeatherMap API (or any weather API used)

## How to Use
1. Open `weatherApp.html` in your browser.
2. Enter the name of the city in the search input.
3. Press **Search** button.
4. View the current weather details for the city.
